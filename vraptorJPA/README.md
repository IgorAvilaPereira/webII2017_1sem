# VRaptor + JPA project

Este é um projeto criado para facilitar utilizar o framework VRaptor com o JPA. Você pode facilmente importá-lo na sua IDE favorita como um projeto Maven.

Após criar seu projeto você pode rodá-lo com um tomcat7 ou +:

```
mvn tomcat7:run
```

Cuidado para *jamais* executar `mvn tomcat:run` pois ele usará um tomcat6 (incompatível).
